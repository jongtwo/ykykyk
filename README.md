# 버티나무 앱 프로토타입

발표용 모바일 웹 프로토타입입니다.

## 로컬 실행
```
index.html을 브라우저에서 바로 열면 됩니다.
```

## Vercel 배포
1. 이 폴더 전체를 GitHub에 push
2. Vercel에서 해당 repo를 import
3. Framework: None (Static) 선택 후 Deploy

## 구현 화면
- **버티맵**: 지도 + 가격 핀 + 가성비 추천 리스트
- **가게 상세**: 사진, 평점, 버티나무 추천 코멘트, 메뉴, 길찾기
- **버티톡**: 1+1 나눔/판매 피드, 지도 미니뷰
- **버티자(마이페이지)**: 프로필, 아낀 금액, 트리 레벨, 관심 가게
